driver.js - Drives all major reocurring site functions including title bars and pictures
JavaScriptEmbed.exe - Converts Embedded.html into a format used in driver.js. This program is run manually offserver by dragging the file you want to convert onto the program.
Output.txt - Stores the output from JavaScriptEmbed.exe so it can be reviewed manually. Code in this file is not directly used on the site.

..\todo.js - Drives the todo list.
